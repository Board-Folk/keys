# BBC Master 128K (Full Size) Replica Keyboard

![Computer Image of BBC Master 128K (Full Size) Replica Keyboard board, front, not much to see than holes](https://github.com/Board-Folk/keys/blob/main/BBC%20Master/Keyboard/images/keyboard3dview_front.png)

A replica keyboard for the full size BBC Master (not Compact). We believe this has been build successfully. Most of the tracks are on the back, former wire links now on the front as tracks.

![Computer Image of BBC Master 128K (Full Size) Replica Keyboard board, rear, where most of the tracks are](https://github.com/Board-Folk/keys/blob/main/BBC%20Master/Keyboard/images/keyboard3dview_rear.png)
